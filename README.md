#  Trenter — Цифровая тетрадь фитнес-тренера
[САЙТ](https://tapokn.github.io/tr-enter/)
### О проекте (сайт)

Этот репозиторий содержит исходный код лендинга для веб-приложения **Trenter** (tr-enter/ttrer/trer) — специализированного инструмента для фитнес-тренеров, работающих индивидуально.

**Trenter** — это простая и прозрачная замена Google Календарю, Excel-таблицам и разрозненным заметкам. Мы не создаём ещё одну перегруженную CRM. Мы даём тренеру «цифровую тетрадь», в которой удобно вести учёт клиентов, тренировок и финансов.

Сайт разработан в рамках практического задания по курсу «Основы фронтенд-разработки» и полностью соответствует методологии **Pure.css** — минималистичного CSS-фреймворка, идеально подходящего для быстрых и чистых интерфейсов. Перед разработкой сайта был проведен [конкурентный анализ](about_project\competitive_analysis.pdf).

### 🎯 Целевая аудитория

- Персональные фитнес-тренеры (самозанятые)
- Инструкторы, работающие соло или в малых группах
- Тренеры, уставшие от сложных CRM и ручного учёта

### 📌 Ключевая ценность продукта

| Что мы даём | Зачем это тренеру |
|-------------|-------------------|
| **Интерактивное демо (Sandbox) без регистрации** | Увидеть продукт в действии за 10 секунд, не заполняя формы |
| **Учёт клиентов и абонементов** | Никаких забытых записей и потерянных контактов |
| **Счётчик оставшихся тренировок** | Всегда точное понимание, сколько занятий оплачено |
| **История платежей** | Прозрачные финансы без сложных отчётов |
| **Синхронизация с Google/Яндекс.Календарём** | Расписание в привычном месте |
| **Вход через Telegram** | Мгновенный старт без паролей |

### 🔍 Ключевая UX-фишка: интерактивное демо (Sandbox) [в разработке...]

Вместо статичных скриншотов мы разместили на первом экране **интерактивный виджет**, имитирующий работу приложения. Пользователь может:

1. Нажать «Добавить клиента» — увидеть форму
2. Ввести имя — увидеть, как появляется карточка
3. Нажать «Провести тренировку» — увидеть, как уменьшается счётчик
4. Нажать «Посмотреть доход» — увидеть сумму

**Зачем:** Это снимает главный барьер конкурентов — «непонятно, что внутри, пока не зарегистрируешься». Тренер доверяет тому, что может «пощупать».

### 📱 Адаптивность (mobile-first)

Сайт полностью адаптирован под все устройства:

| Устройство | Особенности реализации |
|------------|----------------------|
| **Мобильные (320px+)** | Бургер-меню, вертикальная сетка карточек, увеличенные touch-области |
| **Планшеты (768px+)** | 2-колоночная сетка, оптимизированные отступы |
| **Десктоп (1024px+)** | 3-колоночная сетка, горизонтальная навигация, интерактивное демо в полный рост |

Pure.css обеспечивает адаптивную сетку «из коробки» с помощью классов `pure-g` и `pure-u-*`.
